# Personal Task Manager Console App

simple task manager i made for my class project to help organize daily tasks

## what it does

- add tasks with title and description
- view all your tasks
- update task status from pending to completed 
- delete tasks (asks for confirmation)
- sets due date automatically to 7 days

## how to run

you need C# compiler installed then:

```
csc Program.cs Task.cs
```

then run it:
```
Program.exe
```

## how to use

theres a menu when you start it with these options:
1. Add a Task
2. View Tasks  
3. Update Task Status
4. Delete a Task
5. Exit
